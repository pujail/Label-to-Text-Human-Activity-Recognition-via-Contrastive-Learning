# Label-to-Text-Human-Activity-Recognition-via-Contrastive-Learning

**File Description:**
- `NEURA_BLESS.ipynb`: Data Merge, Preprocessing, and Modeling.
- `requirements.txt`: List of required Python libraries.

**How ​​to Run:**
1. Open the `.ipynb` file in Google Colab.
2. Ensure the 'Empatica' and 'Labels' data have been mounted from Google Drive to `/content/drive/MyDrive/ABCASATEAM/`.
3. Run all cells (Run All).

A label-free sensor-to-text framework for translating wearable sensor signals into structured human activity task plans using contrastive alignment and semantic compression.
